<hr>
\section secInstallUG4Boost Boost
<hr>

<ul>
<li> <b>Boost</b>: Download latest release of Boost from "www.boost.org".
<li> Copy the library to some folder on your computer. The folder is arbitrary. 
	(e.g. home/lib/boost_1_46_0)
<li> Add the path of the folder, where boost is located, to the environment 
	variable PATH
	(e.g. 	export BOOST_ROOT = $HOME/lib/boost_1_46_0)
</ul> 

<hr>
\section secInstallUG4CMake CMake
<hr>

<ul>
<li> <b>CMake</b>: Download Cmake from "www.cmake.org" (min Release 2.6)
<li> Install cmake on your computer. Install the command line tools, when requested 
	during installation.
</ul>

<hr>
\section secInstallUG4UG4 ug4

</ul>
